We can take advantage of libcurl straight from within Java programs using this binding.

All you need to do is to get the 'curl-java' package and use it!

Download
The original download places have all died and vanished. The source code seems to have been rescued and is available here:

curl-java

Note
The 'javacurl' project hosted at sourceforge is generally deemed to be crap. Don't use that.
